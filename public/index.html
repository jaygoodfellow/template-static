Modernzr

Modernizr is a JavaScript library that detects the availability of features that stem from the HTML5 and CSS3 specifications. Modernizr does feature detection to discern what the various browsers can and cannot do.

HTML5shiv

HTML5Shiv is a JavaScript workaround to enable styling of HTML5 elements in versions of Internet Explorer prior to version 9, which do not allow unknown elements to be styled without JavaScript.

Learn More »

Browserify

Browserify lets you require modules and other scripts by bundling up all of your dependencies and compiling node-flavored commonjs modules for the browser. It allows npm to be used in the front or backend alike.
